JSSpec: http://code.google.com/p/jsspec/

Install: http://jsspec.googlecode.com/files/JSSpec20080924.zip

JSSpec Server: http://github.com/pivotal/js-spec-server/tree/master
 
  sudo gem install js_spec

  > js_spec_server /path/to/your/javascript/spec/files /path/to/your/javascript/implementation/files

Server is now available at: http://localhost:8080/specs
